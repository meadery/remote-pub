# Ye ~Olde~ Moderne Remote Pub
Web app for running a remote pub.

Currently running at: https://the-ponderous-snail.herokuapp.com/

## Development
Built with typescript on node.

### Installation
1. clone repo
2. `npm install`

### Running locally
`npm run dev`

### Testing
`npm test && npm run lint`
if linting fails:
`npm run lint-fix`
